</div>     
  
>将基于ra2引擎的mod转换成尤里的复仇可以用的版本  
>**⚠️警告：此程序没有自主打开mix的功能，需要手动从mix里把文件提取出来**  
>暂未实现减小程序体积，所以导出的exe体积将有94mb  

## 使用说明  
>导出成exe  
>更改导出目录的**配置文件.ini**
>>**⚠️警告：必须指定mod路径，确保mod路径为红色警戒2mod提取出的文件位置**  
>>**⚠️警告：复制的文件列表请填写红色警戒2文件名称，不要填写尤里的复仇文件名称**  
>>**⚠️警告：一次处理量的值越大，程序越消耗性能**  
>>
>直接运行导出的exe
